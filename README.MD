# COYS Quiz: Tottenham Fan Test

## 프로젝트 소개

이 프로젝트는 오픈소스소프트웨어실습 중간고사 대체 과제로 제작한 Streamlit 기반 웹 애플리케이션입니다.

주제는 토트넘 홋스퍼 팬 지식 퀴즈입니다. 사용자는 로그인 후 토트넘과 관련된 퀴즈를 풀고, 최종 점수에 따라 팬 등급을 확인할 수 있습니다.

## 제출자 정보

- 학번: 2022204068
- 이름: 최인하

## 주요 기능

1. 첫 화면에 학번과 이름 표시
2. 로그인 기능
3. Streamlit 캐싱 기능 적용
4. 토트넘 퀴즈 풀이 기능
5. 점수 계산 및 결과 확인 기능

## 로그인 정보

- ID: spurs
- PW: 1882

## 캐싱 기능 설명

퀴즈 데이터는 앱이 실행될 때마다 반복적으로 새로 생성할 필요가 없기 때문에 `@st.cache_data`를 사용했습니다.

이를 통해 한 번 불러온 퀴즈 데이터를 재사용할 수 있으며, 불필요한 반복 처리를 줄일 수 있습니다.
